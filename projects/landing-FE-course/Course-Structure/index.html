проекти, які ми будемо кодити

Структура курсу

1-й проєкт

  • ви долаєте свої перші кроки у верстанні за детальними підказками. Рухаючись макетом, ви вивчаєте базові HTML теги й основні CSS правила
  • робота з текстом
  • робота з простою графікою (растровою й векторною)
  • робота з кольорами і фігурами
  • просте позиціонування об'єктів

Наприкінці роботи з першим проектом ви ознайомитесь із базою верстання. Вам стане зрозуміло чи цікава вам тематика зтворення сайтів і чи варто вивчати front end більш глибоко.

2-й, 3-й, 4-й проєкти

  • верстка й адаптацію під мобільні пристрої
  • робота з псевдо елементами
  • складне позиціонування за допомогою flex та в інші способи
  • підключення скриптів та анімації
  • командна робота з системою контроля версій git

Навчальні матеріали

  • статті з покроковими підказками
  • відео підказки
  • приклади коду

Наприкінці курсу у вас буде

  • повне уявлення про роботу верстальника і невеличке портфоліо з прикладами кода
  • ви здобудете достатній рівень знання HTML і CSS для подальшого вивчення JavaScript чи основ дизайну
  • у вас сформується команда, всередині якої ви зможете ділитись корисною інформацією і допомогати один одному
